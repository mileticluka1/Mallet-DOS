# Mallet-DOS
Multithreaded DOS tool written in ruby.

# Usage:

`ruby mallet.rb -u https://website.com -t 900`

# Requirements:

Gems: `net/http`, `thread` and `optparse`

1. gem install net-http
2. gem install thread
3. gem install optparse
